
## Git
- 初次設定git
  - `git config --global user.name "johnhuang"`
  - `get config --global user.email "johnhuang1225@gmail.com"`
- 檢查設定
  - `git config user.name`
- 查看幫助
  - `git help <command>`
  
## Create a new repo
- `cd working_dir`
- `git init`
- `git add .`
- `git commit -m "Initial commit"`
